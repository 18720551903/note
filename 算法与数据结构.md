## 1. 栈

1. 题目

   给定一个括号只包括'(' ,')', [', ']' , {', '}'的字符串,判断字符是否有效.

   有效字符需满足:

   1. 左括号必须用相同类型的右括号闭合
   2. 左括号必须以正确的顺序闭合

   注意字符串可被认为是有效字符串

   示例:

   ```js
   输入"{}" true
   输入"()[]{}" true
   输入"{]" false
   输入"([)])" false
   ```

   

```js
  class Stack{
     constructor () {
       this.arr = [];
     }
     add (val) {
      this.arr.push(val);
     }
     pop () {
      return this.arr.pop();
     }
     getSize () {
       return this.arr.length;
     }
    }
    const isValid = s => {
      const map = {
        '{':'}',
        '(':')',
        '[':']'
      }
      const myStack = new Stack();
      for (const item of s) {
        if (map[item]) {
          myStack.add(item) //[{(
        } else {
          const last = myStack.pop();
          if (item !== map[last]) return false; 
        }
      }
      return myStack.getSize() === 0;
    }
    console.log(isValid('[{()})'));
```
